Equipo Muyalware - Sistema: BioPractice 
Guia de BioPractice.

-Crear una base de datos en Postgres (llamada BioPractice) y ejecutar el script "bioPractice.sql"
-Crear una nueva conexión de Base de datos en Netbeans con la base del punto anterior. 
-Abrir el proyecto en Netbeans y ejecutarlo.
-Al abrir el sistema pide usuario y contraseña, se puede utilizar cualquiera que esté en el script, un usuario (Administrador) registrado es 'directional@ciencias.unam.mx' con contraseña 'fT1tIW'.
-Para entrar como alumno puedes registrarte.
